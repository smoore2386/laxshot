# Marketing Tools Notes

## Web Research
- Use `web_search` + `web_fetch` for competitive intel, industry news, and content research
- Preferred sources: App Store/Play Store reviews, Reddit (r/lacrosse, r/mobileapp), TechCrunch, MobileDevMemo
- Competitor apps to track: TeamSnap, SportsEngine, GameChanger, LeagueApps, Stack Sports

## File Organization
- Campaign briefs → `campaigns/`
- Copy drafts → `copy/`
- Research notes → `research/`
- Brand assets / guidelines → `brand/`
- Content calendar → `content-calendar.md`

## Coordination with Engineering
- Use `sessions_send` to ask `engineering-backend` or `engineering-frontend` about feature status
- Format: "Does the app currently support [X]? When is it expected?"
- Don't make feature claims in copy until engineering confirms

## ASO (App Store Optimization)
- Screenshot copy priorities: feature clarity > emotion > social proof
- Keywords to target: lacrosse team app, lacrosse schedule, lacrosse roster
- Review response time goal: < 48 hours for negative reviews

## Analytics Notes
- Primary KPIs: Downloads, D1/D7/D30 retention, NPS, paid conversion rate
- Update `MEMORY.md` when metrics checkpoints are hit

See `content-calendar.md` for the current calendar.
Format: | Date | Channel | Content Type | Status | Notes |
