# src/

Application source code for LaxShot.

```
src/
└── app/
    └── laxshot/          ← Flutter application (see laxshot/README.md)
        ├── lib/          ← Dart source
        ├── ios/          ← iOS platform project
        ├── macos/        ← macOS platform project
        ├── android/      ← Android platform project
        ├── web/          ← Web platform (limited — camera/ML not supported)
        └── pubspec.yaml  ← Dependencies & app metadata
```

See [app/laxshot/README.md](app/laxshot/README.md) for architecture details, data models, and development instructions.
